## Syntax:

Statement separator-terminator: \n

Secondary separator: `&`

Line continuation: `-`

Package import: `package module`

Class import: `class module`

Procedure/function import: `function module`

Required to access code: `package module`

Inline comments: `!`

Block comments: `!!!`
